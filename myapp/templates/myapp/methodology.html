Classification Model

The classification model was trained on the K-Nearest Neighbours algorithm

The data-set used for training was "load_digits" that comes bundled with the Scikit-learn library

The model was saved using the 'pickle' library

Input

The input image must be of .jpg extension

The algorithm uses the 'Pillow' library edit the image performing the following process in sequence:

1. to crop the image to a square,

2. inverts the colour

3. converts the image to grayscale

4. resizes the to 8 x 8 pixel

5. assigns a value to each pixel according to its brightness

Now the image conforms to the parameters of the images on the "load_digits" dataset and can be analysed on the trained model
